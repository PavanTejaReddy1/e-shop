# E-Shop Product Dashboard (Pallet Assignment)

## Features
- Product listing using MUI DataGrid
- Search functionality
- Category filter
- Sorting by price
- Pagination
- Product details page
- Lazy loading
- Responsive design

## Tech Stack
- React.js
- Material UI
- Tailwind CSS
- Axios

## Note
The provided API had CORS restrictions, so mock data was used while keeping API integration structure ready.
